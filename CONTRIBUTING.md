# Contributing to Free Fire India Web

Thank you for your interest in contributing to this project!

## How to Contribute

1. **Fork the Repository**: Create your own copy of the project.
2. **Create a Branch**: `git checkout -b feature-my-new-feature`.
3. **Commit Changes**: Make sure your code follows the existing style and is optimized for web performance.
4. **Push & Pull Request**: Submit your changes for review!

## Code Guidelines

- **Performance First**: Always test frame-rates when adding new 3D assets.
- **Clean Architecture**: Keep entities (Player, NPC) separate from core game logic (Game.js, World.js).
- **Documentation**: Update the README if you add new controls or features.

Happy Coding! 🚀
